Ana Səhifə Qaydalar
Server Qaydaları
1️⃣ Hörmətli olun. Söyüş və təhqir qadağandır.
2️⃣ Cheat və hack istifadəsi qadağandır.
3️⃣ Oyunçuların əşyalarını icazəsiz götürmək olmaz.
4️⃣ Server və Discord reklamı etmək qadağandır.
5️⃣ Admin və moderator qərarlarına hörmət edin.
6️⃣ Bug tapdıqda rəhbərliyə bildirin.
7️⃣ Serverdə exploit və lag yaratmaq qadağandır.
